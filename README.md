# A2C Finanzas V4

Versión consolidada para GitHub Pages y Supabase.

## Archivos principales
- `index.html`
- `styles.css`
- `config.js`
- `js/app.js`
- `js/supabase.js`
- `js/utils.js`
- `sw.js`

## Instalación
1. Haz copia de seguridad del repositorio.
2. Sustituye todos los archivos por los de esta carpeta.
3. Sube también la carpeta `js`.
4. Mantén en `config.js` tu URL y clave pública de Supabase.
5. Espera a que GitHub Pages publique.
6. Borra el service worker anterior y los datos del sitio.
7. Recarga.

La app muestra un error visible si el arranque falla, evitando quedarse indefinidamente en “Cargando”.
