# A2C Finanzas v21

## Actualización obligatoria
Ejecuta `supabase/v10-editable-crypto.sql` después de haber instalado `v8-crypto-wallet.sql`.

## Cambios
- Compras y pagos cripto editables desde el movimiento.
- Traspasos cripto editables y eliminables desde el historial del recurso.
- Cada hucha, carpeta y objetivo muestra el número de criptomonedas y sus cantidades.
- Historial cripto por hucha, carpeta u objetivo.
- Se elimina el botón independiente «Pagar con cripto».
- Para pagar con cripto, crea un Gasto y selecciona una opción del grupo «Pagar con criptomonedas» en Cuenta / espacio.
